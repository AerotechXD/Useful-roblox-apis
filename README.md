# Useful-roblox-apis
well yea in the name useful roblox apis for anyone to use

## Very useful
Games API: GET https://games.roblox.com/v1/games (Game Info)
Games API: GET https://games.roblox.com/v1/games/multiget-place-details (Place Info)
Economy API: GET https://economy.roblox.com/v2/assets/{id}/details (Asset/Subplace Details)
Badges API: GET https://badges.roblox.com/v1/badges/{id} (Badge Info)
Presence API: POST https://presence.roblox.com/v1/presence/users (User Status)
Develop API: GET https://develop.roblox.com/v1/universes/{id}/places (Universe Subplaces)
Client Settings CDN: GET https://clientsettingscdn.roblox.com/v2/client-version/{platform}/channel/{channel} (Roblox App Version)
